# Project LIFE Vision

## 目的

Project LIFEは、毎週5分で1週間を設計し、毎日迷わず行動できる人生管理OSを作るプロジェクトである。

単に予定を登録・管理するのではなく、仕事、学習、食事、休息、家族時間などを一つの生活の流れとして整えることを目指す。

## PCとスマホの役割

### PCは「1週間を設計する場所」

- 1週間全体を見ながら予定を組み立てる。
- テンプレートや生活ルーティンを使い、毎週5分で予定を作る。
- 生活、仕事、学習のバランスを俯瞰して調整する。

### スマホは「今日を実行する場所」

- 今日の予定と現在の行動をすぐ確認する。
- 次に何をするかで迷わない状態を作る。
- 必要な編集や完了記録を、その場で素早く行う。

## プロダクト原則

- 予定入力の量ではなく、生活設計にかかる時間を減らす。
- 機能を増やす前に、毎週5分で予定を作れるかを確認する。
- 個々の予定ではなく、予定同士がつながる生活の流れを扱う。
- PCで設計した内容をスマホでそのまま実行できる状態を保つ。
- 新機能は、毎日の迷いを減らすかどうかを基準に判断する。

## Roadmap

1. PC・スマホ同期：完了
2. スマホUI完成
3. 生活ルーティン
4. テンプレート強化
5. 実績・習慣管理
